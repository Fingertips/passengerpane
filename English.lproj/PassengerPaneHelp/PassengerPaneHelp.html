Using Passenger preferences

You can use Phusion Passenger to run Ruby on Rails applications on the Apache web server included with Mac OS X.

Preference Explanation
Address The address to use for viewing the Rails application in a web browser, e.g. “myapp.local”.
Folder The folder containing your Rails application files.
Run in… The environment (“Development” or “Production”) you want to run the Rails application in.
Allow custom rewrite rules By default, Passenger uses its own rewrite rules to run Rails applications. Please note that you will have to remove or change the default .htaccess file if you turn this on.
